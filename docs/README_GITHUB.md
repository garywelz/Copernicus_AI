# Copernicus AI - Science Podcast Platform

Copernicus AI is a platform for generating and distributing high-quality science podcasts across various disciplines including Mathematics, Physics, Chemistry, Computer Science, and Biology.

## Overview

This repository contains the web application for the Copernicus AI podcast platform, designed to be deployed on Vercel. The platform showcases a collection of AI-generated science podcasts that explain complex scientific concepts in an accessible way.

## Features

- **Podcast Library**: Browse and listen to podcasts across multiple scientific disciplines
- **Responsive Design**: Optimized for desktop and mobile devices
- **Transcript Access**: Read full transcripts of each podcast episode
- **Show Notes**: Access detailed show notes with references and further reading

## Technology Stack

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Audio**: HTML5 Audio API
- **Deployment**: Vercel

## Podcast Categories

1. **Mathematics**: Topics in pure and applied mathematics
2. **Physics**: Explorations of fundamental and theoretical physics
3. **Chemistry**: Discussions on chemical processes and innovations
4. **Computer Science**: Coverage of algorithms, AI, and computational theory
5. **Biology**: Insights into biological systems and research

## Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/garywelz/Copernicus_AI.git
   cd Copernicus_AI
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

## Deployment

This project is configured for deployment on Vercel. Connect your GitHub repository to Vercel for automatic deployments.
